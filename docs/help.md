# 帮助文档

## 快速入门指南

欢迎使用Pingoo网站分析服务！本帮助文档将指导您如何快速开始使用我们的服务。

### 1. 创建账户

1. 访问 [Pingoo官网](https://pingoo.io)
2. 点击右上角的"注册"按钮
3. 填写邮箱地址和密码
4. 验证邮箱地址
5. 登录您的账户

### 2. 添加网站

1. 登录后进入控制面板
2. 点击"添加网站"按钮
3. 输入网站信息：
   - 网站名称
   - 网站URL
   - 网站描述（可选）
4. 点击"创建"按钮

### 3. 安装追踪代码

1. 在网站列表中找到您的网站
2. 点击"获取追踪代码"
3. 复制提供的JavaScript代码
4. 将代码添加到您网站的`<head>`标签中
5. 保存并发布您的网站

### 4. 验证安装

1. 访问您的网站
2. 返回Pingoo控制面板
3. 查看"实时访问"页面
4. 如果看到访问数据，说明安装成功

## 功能详解

### 仪表板概览

#### 关键指标
- **访问量：** 网站的总访问次数
- **独立访客：** 不重复的访客数量
- **页面浏览量：** 所有页面的总浏览次数
- **跳出率：** 只浏览一个页面就离开的访问比例
- **平均会话时长：** 访客在网站上的平均停留时间

#### 时间范围选择
- 今天
- 昨天
- 最近7天
- 最近30天
- 自定义日期范围

### 实时分析

#### 实时访问者
- 当前在线访客数量
- 访客地理位置
- 正在浏览的页面
- 访问来源

#### 实时页面
- 最受欢迎的页面
- 页面访问趋势
- 页面停留时间

### 访客分析

#### 地理位置
- 按国家/地区分布

- 地图可视化展示

#### 设备和浏览器

- 浏览器类型

- 屏幕分辨率

#### 访客行为
- 新访客 vs 回访客

- 会话时长分布

### 页面分析

#### 页面性能
- 最受欢迎页面

#### 退出页面

## 常见问题解答

### 安装问题

**Q: 追踪代码安装后多久能看到数据？**
A: 通常几分钟内就能看到数据，但有时可能需要15-30分钟。如果超过1小时仍无数据，请检查代码安装是否正确。

**Q: 代码应该放在网页的哪个位置？**
A: 建议将代码放在`<head>`标签内，这样可以确保在页面加载时就开始追踪。

**Q: 可以在单页应用(SPA)中使用吗？**
A: 可以！我们提供专门的SPA支持，需要在路由变化时手动调用页面追踪函数。

### 数据准确性

**Q: 为什么我的数据与其他分析工具不一致？**
A: 不同工具使用不同的追踪方法和算法，数据差异是正常的。我们的目标是提供准确的趋势分析，而不是绝对精确的数字。

**Q: 如何排除内部访问？**
A: 可以在设置中添加IP地址过滤，排除公司或个人的IP地址。

**Q: 可以追踪子域名吗？**
A: 可以，您可以在网站设置中配置子域名追踪。

### 隐私和合规

**Q: Pingoo是否符合GDPR？**
A: 是的，Pingoo设计时就考虑了GDPR合规性，不收集个人身份信息，提供数据导出和删除功能。

**Q: 需要显示Cookie同意横幅吗？**
A: 通常不需要，因为Pingoo不使用追踪Cookie。但建议咨询法律专业人士以确保符合当地法规。

**Q: 数据存储在哪里？**
A: 数据存储在安全的服务器上，具体位置取决于您的选择。我们提供多个数据中心选项。

### 技术问题

**Q: 追踪代码会影响网站性能吗？**
A: 我们的代码经过优化，文件大小小于5KB，对网站性能影响极小。

**Q: 支持哪些浏览器？**
A: 支持所有现代浏览器，包括Chrome、Firefox、Safari、Edge等。

**Q: 可以自定义追踪代码吗？**
A: 可以，我们提供自定义配置选项，但建议保持标准配置以确保数据准确性。

### 账户和计费

**Q: 有免费计划吗？**
A: 是的，我们提供功能完整的免费计划，适合个人和小型网站使用。

**Q: 如何升级我的计划？**
A: 在账户设置中选择"升级计划"，选择适合的计划并完成付款即可。

**Q: 可以随时取消订阅吗？**
A: 可以，您可以随时取消订阅，取消后仍可使用到当前计费周期结束。

## 故障排除

### 数据不显示

1. **检查代码安装**
   - 确认代码已正确添加到网页
   - 检查是否有JavaScript错误
   - 验证代码是否被浏览器加载

2. **验证网站设置**
   - 确认网站URL设置正确
   - 检查是否启用了追踪
   - 验证时区设置

3. **浏览器兼容性**
   - 测试不同浏览器
   - 禁用浏览器扩展
   - 检查隐私设置

### 数据异常

1. **流量突然增加**
   - 检查是否有机器人访问
   - 查看流量来源
   - 分析访问模式

2. **流量突然减少**
   - 检查代码是否被移除
     - 验证网站是否正常访问
   - 查看是否有技术问题

3. **地理位置不准确**
   - 了解IP地理定位的限制
   - 检查VPN使用情况
   - 考虑使用CDN的影响

### 性能问题

1. **仪表板加载慢**
   - 清除浏览器缓存
   - 检查网络连接
   - 尝试不同浏览器

2. **实时数据延迟**
   - 了解实时处理的限制
   - 检查服务器状态
   - 联系技术支持
